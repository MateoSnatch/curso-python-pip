# projectPythonCourse 

Cómo ejecutar el proyecto projectPythonCourse

```sh
git clone 
cd projectPythonCourse
sudo apt install -y python3-venv #(si no se tiene aun instalado)
python3 -m venv env
source env/bin/activate
pip3 install -r requirements.txt
python3 main.py 
```
